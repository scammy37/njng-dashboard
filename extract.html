💻 First time on a new computer? Click to expand setup instructions

Options A and B work from any browser with no setup. Only Option C (the script) requires the repo cloned locally with Python installed. Follow these steps once per machine.

Step 1 — Install Git (if not already installed)
1
Go to git-scm.com/download/win and download Git for Windows
2
Run the installer — default options are fine
3
Verify: open PowerShell and run git --version — should print a version number
Step 2 — Install Python (if not already installed)
1
Go to python.org/downloads and download the latest Python 3.x installer
2
Run the installer — check "Add Python to PATH" before clicking Install
3
Verify: open PowerShell and run python --version — should print a version number
Step 3 — Clone the repo
1
Open PowerShell and navigate to where you want the folder:
cd C:\Claude (or wherever you prefer)
2
Clone the repo:
git clone https://github.com/scammy37/njng-dashboard.git
3
Enter the folder:
cd njng-dashboard
Step 4 — Configure Git identity (first time only)
1
Run:
git config --global user.name "Michael Zagame"
git config --global user.email "michael.zagame@gmail.com"
2
When you first push, GitHub will ask you to log in — use your GitHub credentials or a personal access token
Step 5 — Test it
1
From inside the njng-dashboard folder, run:
python serve.py
2
Open http://localhost:8080 — the dashboard should load with your live data
3
You're ready to use Option C
💡 Work PC note: some corporate networks block GitHub or require a VPN. If git clone fails, try connecting via VPN first or use Options A or B which only need a browser.

📥 Option A — Drop the PDF + Ask Claude Easiest

No login automation involved — you save the bill PDF yourself once a month, upload it to the repo, and ask Claude to do the rest.

1
Download the latest bill PDF from njng.com (My Account → Billing → View Bill)
2
Go to github.com/scammy37/njng-dashboard → incoming/ and upload the PDF (works from the GitHub mobile app or any browser — no computer needed)
3
Message Claude (a Claude Code session with access to this repo): "process the new bill"
4
Claude parses the PDF, adds the entry to data.json, commits, and deletes the processed PDF — reload the dashboard in ~1 minute
💡 If Claude can't run the parser for some reason, it can still read the PDF directly and hand you the JSON — fall back to Option B below.

💬 Option B — Use Claude Any device

Works on your phone, tablet, or any computer. Go to claude.ai, attach your NJNG gas bill PDF, and paste this prompt:

Extract the billing data from this New Jersey Natural Gas (NJNG) bill and return a single JSON object in this exact format — no markdown, no explanation: { "label": "Mon YY", "period": "Mon DD–Mon DD, YYYY", "days": 31, "ccf": 63, "therms": 66.72, "cost": 128.44, "supplyCost": 30.47, "deliveryCost": 85.97, "customerCharge": 12.00, "temp": 45 } Rules: - label = start month of billing period, 2-digit year (e.g. "May 26" for a May–Jun bill) - days = integer, number of days in the service period - ccf = integer, "100'S OF CUBIC FEET USED (CCF)" from the meter reading row - therms = number, "BILLING UNITS USED (THERMS)" from the meter reading row - cost = number, 2 decimal places — "THIS SERVICE PERIOD GAS CHARGES" only (exclude NJR Home Services warranty charges, sales tax, or prior balance/payment lines) - supplyCost = the "BGS" line from the Bill Calculation section - deliveryCost = sum of all "DEL" lines from the Bill Calculation section (there may be more than one if the rate changed mid-cycle) - customerCharge = the flat "Customer Charge" line - temp = "Avg Temp This Month" if a numeric value is shown, otherwise null
Then commit it — pick whichever fits where you are:
1
Copy the JSON entry Claude returns
2
From any device (phone, tablet, work PC): go to github.com/scammy37/njng-dashboard → data.json, click ✏️, paste the entry, click Commit changes

From your home PC: open C:\Claude\njng-dashboard\data.json, paste the entry, save, then run git add data.json && git commit -m "Add [Month] bill" && git push
3
Dashboard updates within ~1 minute — reload the page

⚡ Option C — Run the Script Home PC only

Fastest when you're at your home PC. Two ways to do it, both update data.json, commit, and push in one step:

Straight from the PDF (no Claude needed):
1
Open PowerShell in C:\Claude\njng-dashboard (right-click folder → Open in Terminal)
2
Run: python scripts/process_bill.py path\to\bill.pdf
Or from Claude's JSON (Option B):
1
Get the JSON entry from Claude using the Option B prompt
2
Run: python add_bill.py — paste the JSON and press Enter twice
3
Review the values, press Y to commit and push — done

📋 data.json Entry Format

Each bill is one line. Your most recent entry for reference:

{"label":"May 26","period":"May 07–Jun 08, 2026","days":32,"ccf":35,"therms":37.04,"cost":76.38,"supplyCost":16.92,"deliveryCost":47.46,"customerCharge":12.00,"temp":0}
💡 temp: If your bill shows no numeric average temperature (NJNG sometimes leaves it blank for the current month), set it to null.

💡 cost: Use "This Service Period Gas Charges" only — exclude the NJR Home Services warranty line and sales tax, those aren't gas usage.

💡 deliveryCost: If NJNG changed rates mid-cycle you'll see two "DEL" lines in the Bill Calculation box — add them together.
